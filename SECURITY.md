# Security Policy

## Reporting
Email: security@civicpulse.go.id

## Supported Versions
| Version | Supported |
|---------|-----------|
| 1.0.x | ✅ |
| < 1.0 | ❌ |

## Security Measures
- bcrypt password hashing
- JWT with short expiration
- Input validation on all endpoints
- SQL injection prevention via ORM
- Rate limiting
- CORS configuration
